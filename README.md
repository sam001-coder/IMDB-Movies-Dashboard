## Project Overview

This project presents an interactive dashboard built on the IMDB movies dataset
to analyze movie performance, ratings, genres, and revenue trends.

The dashboard was created using Power BI to transform raw movie data
into meaningful visual insights.

---

## Tools Used

- Power BI
- Excel / CSV Dataset
- Data Visualization Techniques

---

## Dashboard Features

- Total Movies KPI
- Average Rating KPI
- Total Revenue KPI
- Total Votes KPI
- Genre-wise movie distribution
- Revenue by year analysis
- Total movies released by year
- Rating distribution histogram
- Interactive slicers (Votes, Rating, Revenue, Year)

---
